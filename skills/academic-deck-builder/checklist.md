# Academic Deck Builder - Quality Checklist

> Use this checklist to verify presentation quality.

---

## Pre-Execution Checks

### Input Validation
- [ ] Presentation topic clear
- [ ] Audience defined
- [ ] Duration specified
- [ ] Source material available

### Preparation
- [ ] Understand audience expectations
- [ ] Know venue requirements
- [ ] Review similar presentations
- [ ] Prepare for Q&A

---

## Structure Checks

### Narrative Arc
- [ ] Clear opening hook
- [ ] Logical problem statement
- [ ] Smooth transitions
- [ ] Strong conclusion
- [ ] Complete story

### Section Balance
- [ ] Motivation appropriate (10-15%)
- [ ] Method detailed (30-40%)
- [ ] Results comprehensive (30-40%)
- [ ] Conclusion concise (10-15%)

### Timing
- [ ] Total time appropriate
- [ ] Sections balanced
- [ ] Buffer time included
- [ ] Q&A time allocated

---

## Content Quality

### Per-Slide Quality
- [ ] One main message per slide
- [ ] Clear headline
- [ ] Supporting evidence
- [ ] Appropriate detail level

### Technical Depth
- [ ] Matches audience expertise
- [ ] Key innovations explained
- [ ] Limitations acknowledged
- [ ] Contributions clear

### Data Presentation
- [ ] Results highlighted
- [ ] Comparisons clear
- [ ] Statistical significance noted
- [ ] Visualizations effective

---

## Visual Design

### Layout
- [ ] Clean and uncluttered
- [ ] Consistent formatting
- [ ] Good whitespace
- [ ] Readable from back

### Figures
- [ ] Architecture diagrams clear
- [ ] Charts well-labeled
- [ ] Colors accessible
- [ ] Resolution sufficient

### Typography
- [ ] Font size readable (≥24pt)
- [ ] Contrast sufficient
- [ ] Consistent fonts
- [ ] Hierarchy clear

### Animations
- [ ] Purposeful (not decorative)
- [ ] Smooth transitions
- [ ] Appropriate speed
- [ ] Enhances understanding

---

## Speaker Notes

### Content
- [ ] Key points covered
- [ ] Timing cues included
- [ ] Transitions scripted
- [ ] Backup content ready

### Delivery
- [ ] Natural language
- [ ] Conversational tone
- [ ] Engagement prompts
- [ ] Confidence builders

---

## Q&A Preparation

### Anticipated Questions
- [ ] 5-10 likely questions
- [ ] Answers prepared
- [ ] Evidence ready
- [ ] Honest about limitations

### Difficult Questions
- [ ] "I don't know" responses
- [ ] Pivot strategies
- [ ] Follow-up promises
- [ ] Graceful handling
