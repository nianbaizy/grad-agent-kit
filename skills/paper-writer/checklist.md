## Output Checks

### Completeness
- [ ] All required files generated
- [ ] All sections present
- [ ] No placeholder content left
- [ ] All TODOs resolved or marked

### Accuracy

- [ ] No fabricated information

### Structure

## Content Quality

### Abstract
- [ ] 150-250 words
- [ ] Clearly states problem
- [ ] Describes method
- [ ] Reports key results
- [ ] Highlights contributions

### Introduction
- [ ] Problem clearly stated
- [ ] Motivation explained
- [ ] Related work briefly reviewed
- [ ] Contributions listed explicitly
- [ ] Paper structure outlined

### Related Work
- [ ] Positions work relative to others
- [ ] Groups related approaches
- [ ] Highlights differences
- [ ] Uses proper citations
- [ ] Identifies gaps

### Method
- [ ] High-level overview provided
- [ ] Technical details explained
- [ ] Figures and equations used
- [ ] Reproducible description
- [ ] Assumptions stated

### Experiments
- [ ] Setup clearly described
- [ ] Datasets specified
- [ ] Baselines defined
- [ ] Results presented clearly
- [ ] Comparisons fair
- [ ] Ablation study included
- [ ] Analysis provided

### Discussion
- [ ] Limitations acknowledged
- [ ] Future work suggested
- [ ] Broader impact discussed
- [ ] Potential concerns addressed

### Conclusion
- [ ] Contributions summarized
- [ ] Key findings restated
- [ ] Significance highlighted
- [ ] Impact statement included
