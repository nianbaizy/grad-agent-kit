# Reviewer Simulator - Quality Checklist

> Use this checklist to verify review quality.

---

## Pre-Review Checks

### Input Validation
- [ ] Paper draft is provided
- [ ] Target venue is specified
- [ ] Review focus is clear
- [ ] All required files available

### Reviewer Preparation
- [ ] Understand venue standards
- [ ] Know acceptance rate
- [ ] Review similar papers
- [ ] Clear of conflicts of interest

---

## Review Process Checks

### Initial Read-Through
- [ ] Read entire paper
- [ ] Understand main claims
- [ ] Note initial impressions
- [ ] Identify key contributions

### Summary Assessment
- [ ] Can summarize in own words
- [ ] Core contributions identified
- [ ] Strengths noted
- [ ] Weaknesses noted

---

## Content Quality

### Novelty Assessment
- [ ] Compared with prior work
- [ ] Originality evaluated
- [ ] Incremental vs. significant assessed
- [ ] Overlap with existing work checked

### Methodology Review
- [ ] Soundness evaluated
- [ ] Assumptions checked
- [ ] Technical correctness verified
- [ ] Theoretical claims validated

### Experimental Validation
- [ ] Setup is appropriate
- [ ] Baselines are fair
- [ ] Results are significant
- [ ] Ablation is comprehensive

### Writing Quality
- [ ] Clarity assessed
- [ ] Organization checked
- [ ] Grammar reviewed
- [ ] Figures/tables evaluated

---

## Review Structure

### Summary
- [ ] 2-3 paragraphs
- [ ] Captures main contributions
- [ ] Fair representation
- [ ] Clear and concise

### Strengths
- [ ] 3+ strengths identified
- [ ] Specific evidence provided
- [ ] Tied to content
- [ ] Genuine strengths

### Major Concerns
- [ ] 2-4 major concerns
- [ ] Specific and detailed
- [ ] Evidence provided
- [ ] Suggestions included

### Minor Concerns
- [ ] 3-5 minor issues
- [ ] Specific but brief
- [ ] Actionable
- [ ] Not nitpicking

### Questions
- [ ] 3-5 questions
- [ ] Relevant to acceptance
- [ ] Not rhetorical
- [ ] Author can answer

---

## Fairness Checks

### Bias Detection
- [ ] No personal bias
- [ ] No institutional bias
- [ ] No methodology bias
- [ ] No topic bias

### Consistency
- [ ] Standards consistent
- [ ] Criteria applied equally
- [ ] Similar papers treated same
- [ ] No double standards

### Constructiveness
- [ ] Tone is professional
- [ ] Criticism is constructive
- [ ] Suggestions are helpful
- [ ] Encouraging where appropriate

---

## Specificity Checks

### Evidence-Based
- [ ] Issues tied to content
- [ ] Page/line references
- [ ] Specific examples
- [ ] Not vague complaints

### Actionable Feedback
- [ ] Clear suggestions
- [ ] Feasible fixes
- [ ] Prioritized
- [ ] Realistic timeline

---

## Completeness Checks

### Coverage
- [ ] All sections reviewed
- [ ] No major gaps
- [ ] Key claims evaluated
- [ ] Limitations identified

### Depth
- [ ] Issues explored thoroughly
- [ ] Root causes identified
- [ ] Implications understood
- [ ] Alternatives considered

---

### Review Report
- [ ] Well-structured
- [ ] Clear organization
- [ ] Professional tone
- [ ] Complete coverage

### Strengths Document
- [ ] Genuine strengths
- [ ] Specific evidence
- [ ] Well-articulated
- [ ] Helpful for authors

### Concerns Document
- [ ] Major concerns clear
- [ ] Minor concerns noted
- [ ] Suggestions provided
- [ ] Prioritized

### Questions Document

- [ ] Author can answer
- [ ] Not rhetorical

- [ ] Actionable
- [ ] Prioritized
- [ ] Realistic
- [ ] Helpful

- [ ] Thorough and specific
- [ ] Constructive and helpful
- [ ] Professional throughout
